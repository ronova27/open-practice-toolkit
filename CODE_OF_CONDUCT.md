# Code Of Conduct

OpenPractice Toolkit should be a welcoming project for musicians, educators, students, researchers, and developers at different skill levels.

## Expected Behavior

- Be respectful and patient.
- Assume good intent, especially with beginner questions.
- Keep critique specific, constructive, and relevant to the work.
- Make room for different musical backgrounds, accessibility needs, and learning paths.
- Respect privacy when discussing recordings, videos, or practice data.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive comments about musical level, language, background, disability, or identity.
- Publishing private information without consent.
- Repeatedly derailing issues or reviews.
- Submitting malicious code, hidden tracking, or secret exfiltration.

## Reporting

For now, report conduct concerns through a GitHub issue if it is safe to do so. If private reporting is needed, wait until the project enables a private maintainer contact path or GitHub private vulnerability reporting.

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code.
