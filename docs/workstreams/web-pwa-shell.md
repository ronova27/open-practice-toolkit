# Web/PWA Shell Workstream

Branch: `workstream/web-pwa-shell`

## Purpose

Plan the first web-first application shell for OpenPractice Toolkit with offline-first behavior, local storage boundaries, and an accessibility baseline.

## Scope

- Propose navigation and module entry points.
- Document PWA and offline-first assumptions.
- Define storage boundaries for practice data and imported media.
- Establish keyboard, screen reader, contrast, and reduced-motion expectations.

## Guardrails

- Do not add a backend or account system.
- Do not add paid infrastructure, secrets, deployments, or analytics.
- Keep the first shell useful and restrained.
