# Product Foundation Workstream

Branch: `workstream/product-foundation`

## Purpose

Refine the founding product materials for OpenPractice Toolkit: README, roadmap, product vision, user stories, glossary, and contribution framing.

## Scope

- Clarify who the project helps and why.
- Keep the subscription-free, open-source, local-first goals explicit.
- Improve beginner-friendly wording without exaggerating current capability.
- Preserve the disclaimer that this is a brand-new founding-stage repository.

## Guardrails

- Do not add application code unless it is a tiny documentation helper.
- Do not add paid services, deployments, secrets, analytics, or account systems.
- Keep changes small enough for focused review.
