# Notation Prehearing Workstream

Branch: `workstream/notation-prehear`

## Purpose

Plan simple notation input and playback so learners can hear notes, chords, and small passages before practicing them.

## Scope

- Compare VexFlow and OpenSheetMusicDisplay for early rendering needs.
- Plan note and chord playback, measure loops, and slow playback.
- Research MIDI and MusicXML export boundaries.
- Keep manual input simple enough for beginners.

## Guardrails

- Do not start a full notation editor.
- Do not add paid infrastructure, secrets, deployments, or analytics.
- Keep export claims tied to what is actually planned or tested.
