# Practice Feedback Workstream

Branch: `workstream/practice-feedback`

## Purpose

Plan local recording and timing feedback for playing, clapping, or tapping against a target rhythm or metronome.

## Scope

- Document recording and onset-detection assumptions.
- Plan timing comparison and latency compensation.
- Define beginner-friendly early/late visual feedback.
- Keep recordings private and local by default.

## Guardrails

- Do not upload recordings or add cloud processing.
- Do not add paid infrastructure, secrets, deployments, or analytics.
- Avoid feedback language that shames beginners.
